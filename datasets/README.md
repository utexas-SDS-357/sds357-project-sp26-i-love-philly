This folder contains two subfolders, input and output. 

The input folder has raw sport schedule and game outcome data. The raw police stop data was too large to upload to GitHub, and was therefore excluded from input.

The output folder contains interim cleaned datasets that were generated by cleaning scripts in the `data_cleaning_and_wrangling folder` and used by Jupyter Notebook files in the `EDA` and `data_cleaning_and_wrangling` folders. The interim datasets created for modeling are not included in this folder, as they are too large to upload to GitHub.
